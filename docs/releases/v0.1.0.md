# Beacon v0.1.0

Beacon의 첫 설치 가능한 로컬 MVP입니다. 프로젝트 파일과 Git을 자동으로 읽어 현재 상태, 부족한 근거, P0–P4 Gate 준비도와 변경 이력을 보여주고 `PROJECT_BOOK.md`로 내보냅니다.

## 설치

```bash
pnpm add --save-dev https://github.com/dayainow/beacon-project-os/releases/download/v0.1.0/dayainow-beacon-0.1.0.tgz
pnpm exec beacon init
pnpm exec beacon open
```

Node.js 22.13 이상이 필요합니다.

## 포함된 흐름

- `beacon init`, `open`, `identity`, `export`
- 파일·Git 자동 관찰과 Beacon Health 신호
- P0–P4 Gate 준비도
- Project Timeline과 SQLite append-only History
- Markdown Project Book
- 단일 번들 CLI와 clean-room 설치 인수 검증

## 현재 제한

- 한 번에 하나의 로컬 프로젝트만 관리합니다.
- 계정, Cloud 동기화와 조직 Portfolio는 포함하지 않습니다.
- Gate 근거를 설명하지만 GO·HOLD·KILL을 자동 결정하지 않습니다.
